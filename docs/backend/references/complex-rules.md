# COMPLEX - 复杂度规则

## COMPLEX-001 文件行数限制

**等级**: 🟡 警告（**新代码**遵循；存量不强制批量拆分）

| 类型 | 建议 | 上限（新代码） |
|------|------|----------------|
| Service | 400 | 600 |
| Entity | 150 | 200 |
| Controller | 300 | 450 |
| DTO/VO | 100 | 150 |

超过上限的新代码必须拆分。

## COMPLEX-002 方法限制

**等级**: 🟡 警告（**新代码**）

| 指标 | 限制 |
|------|------|
| Service 方法 | ≤60 行 |
| Controller 方法 | ≤25 行 |
| 参数数量 | ≤5 |
| 嵌套层级 | ≤3 |
| if-else 分支 | ≤4 |
| 圈复杂度 | ≤8 |
